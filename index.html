Spelling Dictation Practice

  • Provide a name for dictation.Use alphanumeric characters to form the name
  • Add all words to text box to be used for spelling dictation.
  • Words should be space or newline separated and click save.
  • It will clear the text box and cache the words by name
  • The Name(s) will be available in drop down under Practice Dictation using tests



Dictation Words:





  • Select dictation from drop down by name
  • Click dictate which will randomly say the words, each on three times. till all words are said. You can write spelling on paper
  • Once done with dictation,click check to check the words, spellings will appear on screen
  • Tick the check boxes for which you got correct spelling and then press evaluate
  • If you have incorrect words, dictate will be enabled for next practice test using incorrect words and you can repeat till you get all words correct
